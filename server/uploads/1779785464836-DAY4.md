### 时间戳###

1. 使用场景：我们在计算倒计时效果时需使用时间戳。

2. 时间戳就是从某个特定时间到现在的毫秒数

3. 每个时间的时间戳时独一无二的，如果我们要计算从一个时间点到另一个时间点之间的时间，用时间戳想减。

4. 获取时间戳的方法

   ​        `// 1.必须要实例化`

   ​        `const date = new Date()`

   ​        `console.log(date.getTime());`

   ​        `// 2.可以不实例化`

   ​        `console.log(+new Date())`

   ​        `// 3.只能获取当前时间的时间戳`

   ​        `console.log(Date.now())`

## 节点操作##

### DOM节点###

1. DOM节点：dom数里面的节点称为dom节点

2. 分类：元素节点！！是标签

   ​            属性节点，文本节点。。

3. 节点关系：父子关系，兄弟关系，节点关系可以简便程序

4. 父子节点：`son.parentNode`  只能找到亲爸爸

   children属性只获得标签节点

   `li2.nextElementSibling查找下一个兄弟节点`

   `li2.previousElementSibling查找下一个兄弟节点`

### 增加节点###

1. 步骤：创建一个新的节点然后把创建的节点放在指定元素内部

2. 语法：`const box = document.createElement('div')`
           `document.body.appendChild(box)`  追加在父元素的最后一个元素后

3. 追加节点

   `父元素.appendChild(子元素)`追加到父元素的最后一个子元素后

    `父元素.insertBefore(要追加的节点, 追加在哪个节点前)`

   ​

4. 克隆节点： `ul.cloneNode(布尔值)`

   布尔值为true的时候，连同后代节点一起克隆

   布尔值为false的时候，不连同后代节点一起克隆


### 删除节点###

1.删除元素必须通过父元素删除

`父元素.removeChild(子元素)`
